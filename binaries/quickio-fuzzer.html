quickio-fuzzer¶

Synopsis¶

quickio-fuzzer [-p]

Description¶

Runs a QuickIO server configured so that it can be hit by QuickIO’s fuzzer.

Options¶

-p, --profile¶

Run the server in valgrind’s callgrind.
