quickio
=======

.. program:: quickio

Synopsis
--------

quickio [-c /etc/quickio/quickio.ini] [-v]

Description
-----------

QuickIO is a simple, fast, and difficult WebSocket server that focuses on being as quick as possible.

Options
-------

.. option:: --f <config file>, --config-file=<config file>

	Configuration file to load. This option may be given multiple times to load multiple files; it understands most common globbing patterns.

.. option:: -v, --version

	Print the server version and exit.
